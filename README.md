# Weboin Design Studio

A professional, responsive, and production-ready design agency homepage built to showcase creative services.

## Live Demo

[https://weboin-design-studio.vercel.app/](https://weboin-design-studio.vercel.app/)

## Technologies Used

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Key Features

- **Modern Architecture**: Clean, modular component architecture built for scalability.
- **Dynamic UI**: Features a sticky navbar, an immersive hero section, and a curated services grid.
- **Interactive Elements**: Includes a portfolio gallery, an animated contact form, and smooth page transitions.
- **Enhanced UX**: Scroll progress indicator and a dynamic, fully responsive layout.
- **Performance Optimized**: Uses Next.js image caching, priority loading, and SEO best practices for high-tier performance.

## Project Structure

- `/app`: Next.js App Router pages, layout, and global styles.
- `/components`: Reusable UI components (Hero, Portfolio, Services, Contact, Footer, Navbar).
- `/components/ui`: Granular design system elements (Button, SectionHeader, ScrollProgress).
- `/public`: Static assets and imagery.

## Getting Started

First, install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
